Tipos de LAYOUT

Layout Fixed (Fijo)

Este layout tiene un ancho fijo de 800px, independiente del tamaño de la ventana.

Layout Elastic (Elástico)

Este layout usa unidades em (60em) para adaptarse al tamaño de fuente del usuario.

Layout Fluid (Fluido)

Este layout usa porcentajes (80%) para adaptarse al ancho de la ventana.

Layout Híbrido

Este layout combina porcentaje (90%) con valores mínimos y máximos (min-width: 600px, max-width: 1000px).

BOX-SIZING: border-box

Con box-sizing: border-box, el padding y border se incluyen dentro del width especificado (50% en este caso). Sin border-box, el ancho total sería 50% + padding + border.

Como el ciego me tenía gran avaricia de comida, yo, que era muchacho y de buen apetito, no me podía haber con él.

así, viendo que traía todo lo que había de comer en un costal atado con una argolla y candado, determiné de hurtarle.

Y como el fondo del costal estaba ya roto y cosido, yo, cuando me veía solo, descosía un poco cada vez, y sacaba lo que

quería, y luego lo volvía a coser. De esta manera me mantenía, y él no se daba cuenta. Mas, queriendo probar si el vino me

sabía bien, hice un pequeño agujero en el fondo del jarro, que estaba puesto sobre una hornacina, y lo tapé con cera.

Y cuando quería beber, quitaba la cera con el dedo, y ponía la boca al agujero, y el vino corría. Y así me regalaba muchas

veces. Pero un día, queriendo beber, el ciego sintió el jarro más liviano, y alzó la mano al agujero, y sintió el vino correr.

Entonces, llenó el jarro de vino y, esperándome, cuando vine, me dio con él en la cabeza de tal manera que me dejó maltrecho y

sin dientes delanteros.

Elementos FLOTANTES

Float: left

Float: left

originalmente pensada para indicar como se dispone un texto alrededor de una imagen. Hoy se usa para maquetar,

flotando elementos y jugando con sus dimensiones. Sus valores son left o right.

Float: right

originalmente pensada para indicar como se dispone un texto alrededor de una imagen. Hoy se usa para maquetar,

flotando elementos y jugando con sus dimensiones. Sus valores son left o right.
