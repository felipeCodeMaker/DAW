Etiquetas de Estructura

<html>

Ejemplo:

<html> ... </html>

Define el documento HTML. Es el elemento raíz que contiene todo el contenido.

<head>

Ejemplo:

<head> ... </head>

Contiene metadatos sobre el documento, como título, enlaces a CSS, etc.

<body>

Ejemplo:

<body> ... </body>

Contiene todo el contenido visible de la página web.

Etiquetas de Texto

<h1> - <h6>

Ejemplo:

<h1>Título principal</h1>

Este es un H1

Este es un H2

Este es un H3

<p>

Ejemplo:

<p>Este es un párrafo.</p>

Este es un párrafo de ejemplo. Los párrafos se utilizan para agrupar texto relacionado.

<a href="https://ejemplo.com">Enlace de ejemplo</a>

Este es un enlace interno a esta misma sección.
